# What You'll Learn
This flashcard deck covers critical differences between Salesforce relationship types:
Relationship tightness (Loose vs Tight)
Parent record requirements
Sharing and security behavior
Ownership models
Deletion cascading
Roll-up summary field capabilities
Standard object restrictions
Maximum relationship limits per object

# How to Use

# Keyboard Shortcuts
Space or Right Arrow - Flip card or go to next card
Left Arrow - Go to previous card
Enter - Flip current card

# Study Tips
Don't peek! Try to answer each question before flipping the card
Shuffle often - Avoid memorizing the order instead of the concepts
Review multiple times - Repetition is key for retention
Focus on "why" - Understand WHY each relationship works the way it does
Apply to real scenarios - Think about when you'd use each relationship type
